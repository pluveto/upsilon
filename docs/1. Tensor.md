# 张量 (Tensor)

**张量 (Tensor)** 是一个多维数组 ， 它是标量、向量、矩阵的高维拓展。

常见的灰度图像可以用一个矩阵 (2 维张量) 表示；而 RGB 彩色图像则需要用一个 3 维张量表示 (3 个维度分别表示图像的高度、宽度和 RGB 色彩通道)。
